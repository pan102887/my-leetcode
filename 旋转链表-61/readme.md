# 旋转链表
## 描述
给你一个链表的头节点 head ，旋转链表，将链表每个节点向右移动 k 个位置。  

## 示例
### 示例1
![rotate1](../img/rotate1.jpg)  

    输入：head = [1,2,3,4,5], k = 2  
    输出：[4,5,1,2,3]  

![rotate2](../img/roate2.jpg)  

    输入：head = [0,1,2], k = 4  
    输出：[2,0,1]  

## 提示  
- 链表中节点的数目在范围 [0, 500] 内
- -100 <= Node.val <= 100
- 0 <= k <= 2 * 109
